es6之 数组

es6 Arrays

es6 数组主要添加了 map、 reduce、 filter、forEach四个方法，使用过python| java8的朋友对这几个方法应该比较熟悉

一> map

  map()方法返回一个由原数组中的每个元素调用一个指定方法后的返回值组成的新数组。可以通俗的理解为映射，数组与返回结果之间是一一对应的关系。

二> reduce

  reduce()方法接收一个函数作为累加器(accumulator),数组中的每个值(从左到右)开始合并,最终为一个值。

三> filter

  filter()方法使用指定的函数测试所有元素,并创建一个包含所有通过测试的元素的新数组。根据字面理解顾名思义就是过滤器。
