vue webpack

基于vue2使用webpack3.0实现分环境打包

使用vue实际开发过程中,通常需要两或多个环境,测试环境以及正式环境。以公众号开发为例通常开发、迭代、或bug修复都需要在测试环境（测试公众号）确认后才能发布到正式环境。
所以单单使用 npm run build 是绝对不行的，为此我们需要对vue的webpack配置进行一些修改，使得我们的我们的项目聪明点，可以分环境使用不同的打包命令，例如 测试环境使用 npm run test等等
