Hexo主题开发

theme develop

前几天重新捣鼓Hexo theme发现并没有特别合心意的。于是决定花几天时间自己写了一个。

ps：最开始本来打算用vue来写theme，但但发现弊大于利。

    1-> 兼容性问题使用vue就等于放弃了ie8及以下版本的古董浏览器兼容，以后想做兼容是不可能的 这辈子都不可能 除非ie8支持es6… …
    2-> vue项目需要webpack打包不能更这直观开发调试项目，增加开发调试的繁琐程度。感觉有点画蛇添足… …
    3-> hexo支持stylus、sass等动态css语法，ejs能实现组件化。代码本身就可以做到结构清晰，代码复用的目的。

开始项目结构搭建

写代码第一步先把项目结构搭建好,根据Hexo文档theme主要由以下几部分组成

languages

为国际化 让你的网站以不同语言呈现这个不多讲自己看文档

layout

页面布局文件 支持ejs、jade、swing我这边使用的是ejs

2> 在layout目录下新建文件夹_partial 在_partial新建 head.ejs 为html 部分 在前面的layout.ejs中以引入 具体代码如下：

3> 在layout目录下新建index.ejs

4> 修改Hexo项目根目录下_config.yml下theme对象value value为你新建的theme的目录名。启动Hexo项目你的theme初步搭建基本上就已经完成了

source

资源文件 样式文件&静态资源的等存放在source文件夹下

_config.yml

theme配置文件 可以自定义参数 页面通过 theme 参数获得你自己定义的参数值 如：我在_config.yml定义一个 version: 1.0.0 在页面可通过 theme.version 直接调取
